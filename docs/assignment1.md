# The First Week's Assignment
第1週の課題「課題1」について説明します。

___
- ROS 2のTurtleSimで使用されているtopic message types（トピックメッセージ型）を調査し、それぞれの型の用途（使い道／使い所）を説明してください。
    - ROS 1でなくROS 2ですので注意してください。
    - <span style="color: #CC0066;">ROSについての説明やトピック通信についての説明は必要ありません。</span>どんなに書いても点数になりません。
    - topic message types（トピックメッセージ型）はtopic types（トピック型）やmessage types（メッセージ型）と書かれていることもあります。
- 上記に加え、TurtleBotやHSRなど、他のロボットで使用されているtopic message typesについても調査し、考察してくれた人には加点します。ロボティクスコースの人、そして鈴木拓央研究室を希望している人は取り組んでみてください。
- 論文や図書などの文献を参考した場合は、その部分を明記してください。また、AIを利用した場合も、その部分を明記してください。
- レポートの用紙サイズはA4縦とし、レポートの長さは4ページ以内とします。
- 図表を使用しても構いません。
- レポートの作成に使用するソフトウェアはTeXでもWordでもPowerPointでも何でも構いませんが、PDF形式で提出してください。
- レポートのファイル名は「学籍番号_氏名_robotics1.pdf（例：2024311000_鈴木拓央_robotics1.pdf）」でお願いします。
- レポートはTeamsの個人チャット（プライベートチャット）で鈴木拓央＜SUZUKI Takuo＞に提出してください。
- 締切は次回の実験実施日の12時00分までとします。

___
以上です。よろしくお願いします。

［お願い］上述の通り、レポートは個人チャットで提出してもらいますが、ファイルだけ送信するのは印象が悪いと思っています。簡単な文章で十分ですので、「提出いたします」とか「確認をお願いいたします」とか、一文を添えることをオススメします。
